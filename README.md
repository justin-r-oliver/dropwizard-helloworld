# DropWizard Hello World

Simple Dropwizard application demonstrating the following:

* resource serving
* asset bundles
* WAR packaging

See 
https://github.com/rvs-fluid-it/wizard-in-a-box

## Usage

Resource: http://localhost:8080/dropwizard-helloworld-war/api/hello-world
Asset: http://localhost:8080/dropwizard-helloworld-war/static/example.txt
